AndroidSprint — это учебное Android-приложение, предоставляющее список рецептов, распределённых по категориям. Пользователи могут просматривать рецепты, добавлять их в избранное и получать подробную информацию о каждом блюде.

Функции:
- Просмотр списка категорий рецептов;
- Просмотр списка рецептов в выбранной категории;
- Экран с избранными рецептами;
- Детальная информация о рецепте.
  
Технологии:
- Figma — дизайн интерфейса;
- XML — разметка UI;
- Single Activity — архитектурный подход;
- FragmentManager — управление фрагментами;
- Parcelable / Bundle — передача данных между компонентами;
- ViewModel — управление состоянием UI;
- Jetpack Navigation — навигация в приложении;
- OkHttp / Retrofit — работа с сетью;
- Room — локальная база данных.
